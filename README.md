# 🚢 Exploratory Data Analysis (EDA) on Titanic Dataset

## 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on the famous Titanic dataset using Python. The goal is to understand the data, identify patterns, handle missing values, and uncover meaningful insights through statistical analysis and visualizations.

---

## 🎯 Objective
- Understand the structure of the dataset
- Identify missing values and data distributions
- Analyze relationships between different features
- Visualize trends using charts and plots
- Extract meaningful insights from the data

---

## 🛠️ Technologies & Libraries Used
- 🐍 Python
- 🐼 Pandas
- 📊 Matplotlib
- 🎨 Seaborn
- 📓 Jupyter Notebook

---

## 📂 Dataset
**Dataset:** Titanic Dataset (`train.csv`)

The dataset contains passenger information such as:
- Passenger ID
- Survival Status
- Passenger Class
- Name
- Gender
- Age
- Fare
- Cabin
- Embarked Port

---

## 🔍 Exploratory Data Analysis Performed

### ✅ Data Understanding
- Loaded the dataset using Pandas
- Displayed the first five rows using `df.head()`
- Examined dataset information using `df.info()`

### ✅ Statistical Analysis
- Generated descriptive statistics using `df.describe()`
- Identified missing values using `df.isnull().sum()`
- Analyzed survival counts using `value_counts()`

### ✅ Data Visualization
- 📈 Histograms
- 📦 Boxplots
- 🔥 Correlation Heatmap
- 🔗 Pairplots
- 🎯 Scatter Plot

---

## 📊 Key Insights

### 🔹 Missing Values
Missing values were identified in:
- Age
- Cabin
- Embarked

### 🔹 Survival Analysis
- More passengers did not survive than survived.
- Passenger class influenced survival rates.

### 🔹 Fare Distribution
- Fare contains several outliers.
- Ticket prices vary significantly among passengers.

### 🔹 Feature Relationships
- Passenger class and fare show some relationship with survival.
- Visualizations helped identify important patterns and trends.
